# config-for-kitty
idk but this is my config for kitty terminal
## how to start?
### step 1
clone my repository
```sh
  $ git clone https://github.com/SetScript/config-for-kitty/
```
### step 2
rm your kitty config
```sh
  $ rm ~/.config/kitty/kitty.conf
```
### step 3
link my config
```sh
  $ mv ~/config-for-kitty/kitty.conf ~/.config/kitty/kitty.conf
```
